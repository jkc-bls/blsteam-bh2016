Co to jest Front-end?

Tu chcemy opowiedziec ze pojecie frontendu powstalo niedawno. Ze FE + BE to jest separation of concerns: - presentation layer - data access layer

Bardzo dawno temu

  • ubogie, proste strony
  • zawieszający się javascript

Nie tak dawno temu, a nawet teraz

  • flashowe strony
  • ajax - przeładowanie małych fragmentów strony
  • stanowe serwery (stateful servers)
  • cała nawigacja odbywa sie po stronie serwera

Front-end + Back-end

  • FE: aplikacja napisana w Javascript
  • BE: na ogół bezstanowy serwer
  • nawigacja po stronie FE
  • więcej możliwości
  • więcej odpowiedzialności
